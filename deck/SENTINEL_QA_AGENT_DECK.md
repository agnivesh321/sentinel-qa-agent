# Sentinel QA Agent Deck

## Slide 1 - Title

Sentinel QA Agent

AI release guardian for UiPath Test Cloud.

## Slide 2 - Problem

AI agents can create and operate business workflows, but enterprises cannot trust releases without testing, evidence, and human approval.

## Slide 3 - Insight

The winning agent is not the most autonomous agent. It is the agent enterprises can safely put in the release process.

## Slide 4 - Solution

Sentinel QA Agent analyzes a product change, detects security/regression risk, generates UiPath Test Cloud cases, simulates/runs test execution, and blocks unsafe releases for human review.

## Slide 5 - Demo Scenario

AI-assisted invoice refund workflow:

- Payment logic
- Authentication roles
- AI refund recommendation agent
- ERP webhook sync
- Review dashboard state machine

## Slide 6 - UiPath Fit

UiPath Test Cloud owns test execution and evidence.

UiPath Orchestrator runs the automation.

Action Center handles human release review.

## Slide 7 - Differentiation

Not a chatbot. Not a generic test generator.

Sentinel is a release control plane for AI-built software.

## Slide 8 - Results

- Risk score: 96/100
- Decision: block release
- Generated tests: 6
- Failed case: duplicate invoice on checkout retry
- Human approval required

## Slide 9 - Prize Strategy

Targets:

- Best of UiPath Test Cloud

## Slide 10 - Next Steps
